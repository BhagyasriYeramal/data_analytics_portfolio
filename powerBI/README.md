## Power BI Project

This folder contains the Power BI dashboard 
and screenshots.

**Tools used:** Power BI Desktop

**Concepts covered:**
- Power Query (data cleaning and transformation)
- Star Schema data modeling (Fact and Dim tables)
- DAX Measures (Total Sales, YOY Growth %, 
  Avg Days to Ship, Total Orders)
- Interactive Dashboard with slicers
- Relationships between tables
