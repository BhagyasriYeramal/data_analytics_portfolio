## Dataset

This folder contains the dataset used across 
all three tools — SQL, Excel and Power BI.

**Tables:**
- Customers (5 rows)
- Employees (5 rows)
- Products (5 rows)
- Orders (10 rows)
- OrdersArchive (7 rows)

**Source:** Sample sales dataset used for 
learning and practice.


**Note:** orders_combined table was created 
by combining Orders and OrdersArchive tables 
using SQL (see SQL/queries/orders_combined.sql)
