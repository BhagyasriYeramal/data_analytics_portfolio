# Sales Analytics Portfolio Project

## Overview
End-to-end data analytics project built using SQL, 
Excel and Power BI on a sales dataset containing 
customers, employees, products and orders data.

---

## Tools & Technologies
| Tool | Purpose |
|------|---------|
| SQL Server (SSMS) | Data extraction, cleaning, analysis |
| Excel | Data cleaning, analysis, dashboard |
| Power BI | Data modeling, DAX, visualization |

---

## Project Structure

📁 Dataset — Source data tables (CSV files)

📁 SQL — SQL queries
- orders_combined_setup.sql — Created combined orders table
- eda_queries.sql — Exploratory data analysis
- advanced_analytics.sql — Window functions, YOY analysis
- customer_summary.sql — Customer report view
- product_summary.sql — Product report view

📁 Excel — Excel project file and screenshots
- Sales_project.xlsx
- Dashboard screenshots

📁 PowerBI — Power BI project file and screenshots
- Star schema data model
- DAX measures
- Interactive dashboard

---

## Key Skills Demonstrated
- SQL: Joins, CTEs, Views, Window Functions,
  CASE WHEN, Date Functions
- Excel: XLOOKUP, Power Query, Pivot Tables,
  IFS, SUMIFS, Conditional Formatting, Dashboard
- Power BI: Star Schema, DAX Measures,
  YOY Growth, Interactive Dashboard with slicers

---

## Dataset
Sample sales dataset containing:
- 5 Customers
- 5 Employees
- 5 Products
- 17 Orders (2024 + 2025 combined)


**Source:** Dataset from "The Complete SQL Bootcamp" 
course by Baraa Khatib Salkini (Udemy). 
All analysis, queries, Excel work and Power BI 
dashboard were independently built by me using 
this dataset for learning and practice.
