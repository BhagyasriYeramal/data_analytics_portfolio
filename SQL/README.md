## SQL Project

This folder contains SQL queries written for 
the Sales Analytics project.

**Tool used:** SQL Server Management Studio (SSMS)

**Concepts covered:**
- Joins (LEFT JOIN, INNER JOIN)
- CTEs (Common Table Expressions)
- Views
- Aggregate Functions (SUM, COUNT, AVG, MAX, MIN)
- CASE WHEN
- Date Functions (DATEDIFF, GETDATE)
- NULLIF to handle divide by zero
- Subqueries
